  • Internet connectivity

ISP-Internet Service Provider
DNS-Domain Name System
HTTP-Hypertext Transfer Protocol

Homepage is the first page on a website every page contains the following

  • HTML-->Hypertext Markup Language
  • CSS-->Cascading Style Sheet
  • JAVASCRIPT
  • URL-->Uniform Resource Locator

STRUCTURES OF HTML

  1. Tags
    • Container tag (it only has beginning and ending tag)
    • Empty/self close tags (it only has starting tag without ending tag)
  2. Comments:Are not intercepted by the browser < !--HELLO-- >
  3. Attributes:This is used to change the default setting of a tag. The syntax is name="value" attributes can only be used in the opening tag e.g <hr colour = "Red", Size = "50%">
